# MediaOS

## The Contest: What to Build

Build a real tool that delivers value to my media buying team.

The scope is wide. You can build anything, as long as you genuinely believe it's a valuable tool for our media buying operation. Potential ideas include creative generation tools, MCP connectors to ad platforms, automation workflows, performance dashboards, landing page builders, or something else entirely. The bigger the better. Flex your muscles.

## The criteria:

You believe it will solve a real problem for our business.
You can convince me it will solve a real problem for our business.
It actually works — or it's built to the highest level of functionality you could reasonably achieve.
That's it. Build something.

If you don't have direct experience with media buying, do the research. We advertise at scale across all major ad platforms — Google, Meta, Taboola, TikTok. We report on that data to optimize. We build landing pages to generate and collect leads. This simple process requires immense technical input, and there are dozens of ways to add value with AI tooling.
